## Environment Setup

### Docker Command
```bash
sudo docker run --rm \
  --device=/dev/dri \
  --device=/dev/mem \
  --group-add video \
  --privileged \
  --shm-size=8g \
  -e http_proxy=http://proxy.ims.intel.com:911 \
  -e https_proxy=http://proxy.ims.intel.com:911 \
  -v /mnt/disk3/home/lzhang2/hanchao/intel/oneapi:/opt/intel/oneapi:ro \
  -v /mnt/disk3/home/lzhang2/hanchao:/workspace \
  -w /workspace/torch-xpu-ops/test/xpu/distributed \
  intelgpu/ubuntu-26.04-rolling:26.18 bash
```

### Inside Container
```bash
source /opt/intel/oneapi/setvars.sh --force
export CCL_ATL_TRANSPORT=mpi
export ZE_AFFINITY_MASK=0,1,2,3,4,5,6,7  # for 8-GPU
# or ZE_AFFINITY_MASK=0,1,2,3 for 4-GPU
```

### Run Benchmark
```bash
mpirun -n 8 ./bench_ccl_allreduce_c_api --min 12 --max 28 --warmup 20 --loop 100
```
